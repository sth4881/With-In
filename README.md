# 💊 OCR을 활용한 처방전 관리 플랫폼 - With In

## 🎯 주요 기능
- 처방전 촬영하고 OCR을 적용해서 인식한 내용을 바탕으로 **나만의 처방 데이터**를 만들 수 있습니다.
- 건강보험심사평가원에 등록된 데이터를 바탕으로 의약품에 대한 **성분과 효능 및 주의사항** 정보를 제공합니다.
- 병원을 방문할 때마다 처방 데이터를 기록하고 **언제**, **어떤 병원에서**, **어떤 약을** 처방받았는지 조회할 수 있습니다.

## 🛠 사용 기술
- Android(Java)
- Naver CLOVA OCR API
- Postman
- SQLite3
