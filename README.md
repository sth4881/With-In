# 💊 OCR을 활용한 처방전 관리 플랫폼 - With In

## 🎯 주요 기능
- 처방받은 의약품 및 건강보험심사평가원에 등록된 **약의 성분과 효능 및 주의사항**을 제공합니다.
- 처방전 촬영 및 OCR 적용을 통해서 인식한 내용을 바탕으로 **나만의 처방 데이터**를 만들 수 있습니다.
- 병원을 방문할 때마다 처방 데이터를 기록함으로써 **언제**, **어떤 병원에서**, **어떤 약을** 처방받았는지 조회할 수 있습니다.

## 🛠 사용 기술
- Android(Java)
- Naver CLOVA OCR API
- Postman
- SQLite3
